# How to Run the Modern Banking System

Follow these steps to compile and run the project on your local machine.

## 1. Prerequisites

- **Java Development Kit (JDK)**: Ensure you have JDK 8 or a compatible version installed.
- **MySQL Server**: Make sure MySQL Server is running.
- **Database Credentials**: The application is configured to use the following default credentials for MySQL:
  - **Username**: `root`
  - **Password**: `123456`

  The application will automatically create the database (`modern_banking_db`) and tables on the first run.

## 2. Compilation

Open a terminal or command prompt in the project root and compile into the module’s `ModernBankingSystem/bin` folder to keep the layout consistent with the installer and launcher.

Windows (cmd.exe or PowerShell):

```
javac -source 8 -target 8 -d ModernBankingSystem\bin -cp "ModernBankingSystem\lib\*;ModernBankingSystem\src" ModernBankingSystem\src\com\banking\Main.java ModernBankingSystem\src\com\banking\db\*.java ModernBankingSystem\src\com\banking\model\*.java ModernBankingSystem\src\com\banking\controller\*.java ModernBankingSystem\src\com\banking\view\*.java ModernBankingSystem\src\com\banking\util\*.java
```

macOS/Linux (bash/zsh):

```
javac -source 8 -target 8 -d ModernBankingSystem/bin -cp "ModernBankingSystem/lib/*:ModernBankingSystem/src" ModernBankingSystem/src/com/banking/Main.java ModernBankingSystem/src/com/banking/db/*.java ModernBankingSystem/src/com/banking/model/*.java ModernBankingSystem/src/com/banking/controller/*.java ModernBankingSystem/src/com/banking/view/*.java ModernBankingSystem/src/com/banking/util/*.java
```

## 3. Execution

After successful compilation, run the application. On Windows, you can double‑click the launcher or use the command below.

Windows:

```
ModernBankingSystem\run.bat
```

macOS/Linux:

```
java -cp "ModernBankingSystem/bin:ModernBankingSystem/lib/*" com.banking.Main
```

The application window should now appear, starting with the login screen.

## 4. Installer (Optional)

You can also create a professional installer for the Modern Banking System using NSIS. For more information, please see the `howtocreateinstaller.txt` file in the root directory.
